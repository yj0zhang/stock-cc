# stock-cc
股票信息展示，主要是小程序端（小程序没有金融相关资质，只能发体验版）

[点击查看h5](https://yj0zhang.github.io/stock-cc/)

使用TypeScript开发

可能需要全局安装一下taro
```js
// 使用 npm 安装 CLI
$ npm install -g @tarojs/cli
// OR 使用 yarn 安装 CLI
$ yarn global add @tarojs/cli
// OR 安装了 cnpm，使用 cnpm 安装 CLI
$ cnpm install -g @tarojs/cli
```

然后安装一下依赖，接着运行脚本
